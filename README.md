# Web Drum Machine & Loop Pedal

A browser-based music creation tool designed for children ages 8-12.

## Features

- **Drum Machine**: 16-step sequencer with 12-16 drum sounds and preset patterns
- **Loop Pedal**: 4-6 track recording and playback with microphone input
- **Session Management**: Save and export your musical creations
- **Multiple Themes**: Dark, Green-on-Black, and Bold-Vivid color schemes
- **Child-Friendly**: Large buttons, intuitive interface, interactive tutorial

## Quick Start

1. Open `index.html` in a modern web browser (Chrome, Firefox, Safari, or Edge)
2. Allow microphone access when prompted (optional, for loop recording)
3. Start creating music!

## Technology

- Vanilla JavaScript (no frameworks)
- Web Audio API for professional audio processing
- HTML5 Canvas for visualizations
- Fully responsive, works on desktop and tablets

## Browser Support

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## Development

No build process required! Simply open `index.html` in your browser.
